# {{project_name}}

A Love2D game project.

## Getting Started

```bash
# Install dependencies
depot install

# Run the game
love .
# or
depot run dev
```

## Project Structure

- `main.lua` - Main game loop
- `conf.lua` - Love2D configuration
- `src/` - Game source code
- `assets/` - Game assets (images, sounds, etc.)
